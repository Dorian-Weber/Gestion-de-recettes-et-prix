# Gestionnaire de Reccettes et calcul de prix
Ce projet est une application simple pour gérer des recettes culinaires et calculer le coût total des ingrédients nécessaires pour préparer chaque recette.
## Fonctionnalités
- Ajouter, modifier et supprimer des recettes.
- Calculer le coût total des ingrédients pour chaque recette.
- Afficher une liste de toutes les recettes disponibles.
## Installation
1. Clonez le dépôt :
   ```bash
   git clone
   ```
2. Accédez au répertoire du projet :
   ```bash
    cd gestionnaire-recettes
    ```
3. Installez les dépendances :
    ```bash
    npm install
    ```
4. Démarrez l'application :
    ```bash
    npm start
    ```
## Utilisation
- Utilisez l'interface pour ajouter de nouvelles recettes, modifier les existantes ou supprimer celles que vous ne souhaitez plus conserver.
- Pour chaque recette, entrez les ingrédients et leurs coûts pour calculer le prix total.
